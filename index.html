Introduction

The goal of this project was to create a competitive Chess AI that uses advanced algorithms to make optimal moves in real-time, simulating intelligent decision-making. The AI evaluates various possible moves and their outcomes using a blend of strategic algorithms, making it highly effective in gameplay.

Project Overview

In chess AI development, algorithms like Minimax and Alpha-Beta Pruning are crucial. The AI can foresee potential moves and consequences, using Quiescence Search to eliminate noise from certain types of move sequences. To evaluate game positions, the AI applies a scoring system based on piece values and positional strength using piece-square tables.

Algorithms and Techniques

Minimax Algorithm with Alpha-Beta Pruning

The Minimax algorithm evaluates all possible moves in a game tree to find the move that maximizes the player's advantage while minimizing the opponent's potential. Alpha-beta pruning optimizes this by cutting off parts of the tree that don’t need to be explored, speeding up the search without affecting the result.

Figure: Example of a Minimax game tree.

Quiescence Search

The Quiescence search enhances the Minimax algorithm by preventing it from making poor decisions due to the horizon effect (evaluating an unstable position just before it becomes stable). Instead, the AI explores capture moves beyond the normal search depth until a stable or "quiet" position is found.
